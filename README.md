##### 项目简介

本系统旨在通过数字化手段优化社区服务流程，提升管理效率。主要功能涵盖：房产与业主档案管理、自动化费用清算与在线支付、全流程报修调度以及社区动态公告发布。系统采用响应式架构设计，确保在PC端管理后台与移动端业主自服务界面均能提供流畅的操作体验。
目标用户主要分为两类：一是物业管理人员，利用系统实现高效的资产监管、财务核算与工单分派；二是社区住户，通过手机端实现足不出户的账单查询、一键报修及重要通知接收，构建便捷、透明的智慧社区环境。

##### 技术栈

前端：Vue3+ Vite+Element Plus+Pinia+Vue Router+Axios+ECharts

后端：Spring Boot+Spring Security + JWT+MyBatis-Plus

数据库：MySQL

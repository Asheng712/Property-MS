## 我完成的工作

---------

### ### 1. API 设计

**用户认证 API**：设计了基于 JWT 令牌的登录 (/login) 与权限隔离的注册 (/register) 协议。

**业务资源 API**：定义了**五大核心模块**（资产全景、商业合同、智能账单、报修调度、投诉闭环）的全量 RESTful 接口。

**查询接口设计**：规范了复杂的树形结构接口 (/assets/tree)、状态看板接口 (/repairs/kanban) 以及多条件组合分页查询。

### ### 2. 文档编写

**OpenAPI 文档**：编写并维护了完整的 Swagger 3.0 标准文档，涵盖了系统中全部 功能接口。

**API 使用说明**：制定了《前后端交互规范》，明确了 DTO、VO 与 Result 统一封装对象的数据标准。

### ### 3. 后端实现

**API 路由定义**：基于 Spring Boot 3.x 实现了多模块架构下的控制器层，确保了 /api/v1 的规范路径映射。

**业务逻辑处理**：

* 完成了**资产管理**、**权限校验**、**用户体系**的核心业务代码开发。

* 实现了 DTO 到 Entity 的映射转换，并基于 MyBatis-Plus 实现了稳健的持久层逻辑。

**错误处理**：构建了全局异常处理机制 (@RestControllerAdvice)，实现了对 SQL 异常、业务逻辑异常的自动捕获与友好返回。

### ### 4. 测试

**Postman/Apifox 测试集合**：建立了全量接口测试脚本集，覆盖了认证、资产查询、用户注册等核心路径。

**后端单元测试**：针对 UserService 及 HouseService 的核心逻辑编写了 JUnit 测试用例。

**测试用例数量**：2个（登录、注册校验）。

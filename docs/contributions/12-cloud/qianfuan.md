## 心得体会

1. Vercel 作为免费前端托管平台，对 Vite 项目支持完善，几乎零配置即可部署
2. 环境变量管理是云部署的关键步骤，敏感信息不能提交到代码仓库，需通过平台控制台或 Secret 管理
3. SPA 应用在静态托管时需要特别注意路由配置，核心是确保所有路径请求都能正确回退到 `index.html`
4. Docker 部署前端虽然比 Vercel 复杂，但提供更高的灵活性（如自定义 Nginx 规则、API 代理等），适合有自托管需求的场景

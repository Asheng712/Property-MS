##### 前端设计 (Front-end Design)

系统采用 “PC管理端 + 移动端业主端” 双端架构，共享同一套后端API。

1. ###### 响应式布局策略

系统采用 “移动优先 + 栅格系统” 的设计思路：

大屏 (PC管理端): 侧边栏菜单 + 顶部面包屑导航 + 中央工作区。

小屏 (移动端业主端): 底部Tab导航 + 卡片流布局，使用 Vant 4 移动端组件库，适配 320px-428px 手机屏幕。

实现手段: PC端使用 Element Plus 的 el-row / el-col 断点属性（xs, sm, md, lg）；移动端采用 Vant 4 rem 自适应。

2. ###### 页面结构设计

登录页: 支持管理员/业主统一登录。

工作台 (Dashboard):

管理员（PC端）：显示房产总数、报修待处理数、本月收入情况。

业主（移动端）：显示待缴账单、报修进度、最新公告、快捷服务入口。

功能模块: 房产档案、住户信息、缴费中心、在线报修、社区公告。

3. ###### 前端路由权限

PC管理端：动态路由，根据登录角色（Role）动态挂载路由表。

移动端：5个Tab底部导航（首页/账单/服务/公告/我的），路由守卫拦截未登录访问。

双端均通过 Axios 拦截器检测 401/403 后自动跳转登录页。

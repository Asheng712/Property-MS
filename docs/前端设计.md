##### 前端设计 (Front-end Design)

1. ###### 响应式布局策略

系统采用 “移动优先 + 栅格系统” 的设计思路：

大屏 (PC): 侧边栏菜单 + 顶部面包屑导航 + 中央工作区。

小屏 (手机/平板): 侧边栏自动隐藏（变为汉堡菜单抽屉），列表页由表格转为卡片流展示。

实现手段: 使用 Element Plus 的 el-row / el-col 断点属性（xs, sm, md, lg）。

2. ###### 页面结构设计

登录页: 支持管理员/业主切换。

工作台 (Dashboard):

管理员：显示房产总数、报修待处理数、本月收入情况。

住户：显示我的账单、最新公告、报修进度。

功能模块: 房产档案、住户信息、缴费中心、在线报修、社区公告。

3. ###### 前端路由权限

动态路由: 根据登录后的角色（Role），动态挂载路由表。

路由守卫: 拦截未登录或无权限的访问。

# 设计说明：智慧物业管理系统 (WisdomPM)

## 版本
v2.1.1

## 项目概述
前后端分离的智慧物业管理系统，覆盖资产管理、合同租赁、财务缴费、报修投诉、公告发布等核心场景。包含 PC 管理端（物业管理员）和移动端（业主）双端。

## 配色方案
- 主色: #1890FF（科技蓝，主按钮和导航）
- 辅助色: #F0F2F5（浅灰，背景和卡片）
- 强调色: #52C41A（健康绿，缴费成功/状态正常）
- 警告色: #E6A23C（待缴费/待处理）
- 危险色: #F56C6C（已逾期/已驳回）

## 核心页面

### PC 管理端
- **运营看板**: 待收费/欠费数据实时统计、今日投诉/报修、各类型资产租售占比饼图
- **资产管理**: 楼栋→单元→房间树形结构 + 列表 CRUD
- **合同管理**: 合同列表 + 资产申请审批（购买/租赁）
- **账单引擎**: 按费用类型生成账单 + 缴费流水核销 + 物业费单价配置
- **报修调度**: 看板分组 + 工单派发 + 状态跟踪
- **投诉处理**: 列表筛选 + 处理反馈
- **公告发布**: 富文本编辑 + 定向推送

### 移动端
- **首页**: 轮播 + 快捷入口 + 待办事项（待缴费数量/进行中报修/处理中投诉）
- **账单**: 待缴/已缴列表 + 在线缴费
- **报修**: 名下房屋选择 + 提交 + 进度跟踪
- **投诉**: 提交 + 列表 + 结果查看
- **资产**: 购买/租赁申请
- **个人**: 信息编辑 + 关于我们

## 交互规范
- 侧边导航支持折叠，md 以下自动收起
- 列表支持分页 + 关键词搜索 + 状态下拉筛选
- 状态徽标统一色调: 成功=green, 警告=orange, 危险=red, 信息=blue
- 移动端使用底部 Tab + 无限滚动 + 下拉刷新

## 技术架构
- 前端: Vue 3 + TypeScript + Pinia + Vite
- 后端: Spring Boot 3 + MyBatis-Plus + JWT
- 数据库: MySQL 8.4 (14 张核心表)
- 部署: Docker Compose + GitHub Actions CI/CD
